Profile and associated profiles

Each house can have more than one profile that can use this APP. Not all of this profiles will have all the permissions that Adaptative has. The profiles with all permissions can create new profiles with less permissions and edit or delete this profiles with lower permission level.

For example:

Imagine we have an account with a profile without retrictions, or what is the same, with all the permissions given.

That profile creates another two accounts, but this ones only will have permission to see, but not to interact or write in any place of the APP.

Clients

The Clients are all the devices available on our online shop that can be installed in a home following the simple instructions. From this APP, the Clients can be read or written by you, but you also can associate or divide the Clients that you own by groups. This can be done dividing the Clients registered in the APP in the number of Rooms that you need to organice them.

Rooms

Rooms are all the groups or dividers that you have to organice all the Client devices intalled in your home. From the APP you can create, edit or delete rooms or interact with the Clients of each Room, modifiying or reading them.
